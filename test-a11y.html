🔍 Cómo Verificar

  1. Abrir index.html en un navegador
  2. Presionar Tab para navegar - verificar focus visible
  3. Usar NVDA/JAWS/VoiceOver para probar lectores de pantalla
  4. Probar en móvil con diferentes tamaños de pantalla
  5. Activar "Reducir movimiento" en sistema operativo
  6. Activar "Alto contraste" en sistema operativo
  7. Verificar que todas las acciones sean posibles solo con teclado
